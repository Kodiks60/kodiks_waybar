Custom Waybar configuration for your desktop!

## Installation

1. **Clone the repository:**
```bash
git clone https://github.com/Kodiks60/kodiks_waybar
cd kodiks_waybar
```

2. **Run installer for your distro:**

| Distro | Command |
|--------|---------|
| Arch/Manjaro | `./arch.sh` |
| Debian/Ubuntu | `./debian.sh` |
| Fedora | `./fedora.sh` |
| OpenSUSE | `./opensuse.sh` |

```bash
chmod +x *.sh
./your_distro.sh
```

3. **Restart Waybar:**
```bash
pkill waybar
waybar &
```
