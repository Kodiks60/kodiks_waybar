# kodiks waybar
waybar config for you :)

How install him
1. Copy the repository:

```bash
git clone https://github.com/Kodiks60/kodiks_waybar
```
2. Let's setup
If you have arch linux/manjaro or pacman manager. Use arch.sh
```bash
cd kodiks_waybar
chmod +x arch.sh
./arch.sh
```
If you have Debian/Ubuntu/Mint or apt manager. Use debian.sh
```bash
cd kodiks_waybar
chmod +x arch.sh
./debian.sh
```
