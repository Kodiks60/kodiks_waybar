# kodiks waybar
waybar config for you :)

How install him
1. Copy the repository:

```bash
git clone https://github.com/Kodiks60/kodiks_waybar
```
2. Let's setup.
If you have arch linux/manjaro or pacman manager. Use arch.sh
```bash
cd kodiks_waybar
chmod +x arch.sh
./arch.sh
```
If you have Debian/Ubuntu/Mint or apt manager. Use debian.sh
```bash
cd kodiks_waybar
chmod +x debian.sh
./debian.sh
```
If you have Fedora or dnf manager. Use fedora.sh
```bash
cd kodiks_waybar
chmod +x fedora.sh
./fedora.sh
```
If you have OpenSUSE or zypper manager. Use opensuse.sh
```bash
cd kodiks_waybar
chmod +x opensuse.sh
./opensuse.sh
```
3. And let's test out. You need to kill waybar.
```bash
kill waybar
```
And start him again.
```bash
waybar
```
